Könyvtárkezelő program

Ennek a könyvtárkezelő programnak a fő célja, hogy a felhasználó egy helyen kezelhesse saját könyvlistáját.
Fő funkciói:
- Könyv hozzáadása, a felhasználó megadhatja a könyv címét, szerzőjét és kiadás éve adatokat és a "Könyv hozzáadása" gombbal elmentheti a listába
- Könyv törlése, a felhasználó kijelöl egy elemet és a "Kijelölt törlése" gombbal eltávolíthatja
- Fájlba mentés, a felhasználó a "Mentés CSV-be" gomb segítségével elmenti az aktuális könyvlistát a konyvek.csv fájlba
- Fájl betöltése, a program induláskor automatikusan ellenőrzi a CSV fájl meglétét, és ha találja, akkor betölti azokat
